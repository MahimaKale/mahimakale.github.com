Named Function

The named function returns the name of the pet. This is a simple function idea that I got from the pet I am fostereing for the weekend.

           function pet(name){
               return name;
           }
           console.log(pet('Kunami'));

Function Expression

The function Expression creates a variable and is assigned an anonymous function. 

            const pet = function(name){
                return name;
            }
            console.log(pet('Kunami'));

Arrow Function

The variable pet uses arrow function to retune the name of the pet.

            const pet = ('name') => `${name}`
            console.log(pet(Kunami));
